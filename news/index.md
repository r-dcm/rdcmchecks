# Changelog

## rdcmchecks 0.1.1

CRAN release: 2026-03-11

- Fixed issue in
  [`check_data()`](https://rdcmchecks.r-dcm.org/reference/check_data.md)
  where missing responses resulted in an error when `missing` was
  specified.

- [`check_missing()`](https://rdcmchecks.r-dcm.org/reference/check_missing.md)
  can now be used to ensure that a value is a length 1 scalar (of any
  type).

## rdcmchecks 0.1.0

CRAN release: 2025-09-15

- Initial CRAN submission.
